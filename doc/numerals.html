Numerals

;https://github.com/viebel/lambda-calculus (ns demo.lambda (:refer-clojure :exclude [+ - * inc dec]) (:require-macros [lambda-calculus.macros :refer [lambda]]) (:require [lambda-calculus.numerals :refer [num + - * inc dec]]))

A Numeral n is viewed as function that is applied n times:

(num 8)

Comparison

(= (num 8) (num 8))
(= (num 3) (num 8))

Incrementaion

(inc (num 8))

(= (num 3) (inc (num 2)))

Decrementaion

(inc (num 8))

Addition

(+ (num 2) (num 3))

Subtraction

(- (num 7) (num 3))

Multiplication

(* (num 2) (num 3))

Exponentiation

(** (num 2) (num 3))
